４月１２日

Daniell の原論文が、ネットで公開されていました。 １００年以上前に書かれたとは思えぬ現代的かつ明快なものです。
これとほぼ同時期に出版された Caratheodory の本と読み比べてみると、 こちらも今に通用する書き方に驚かされます。
数と集合の話から始まり、p.160 の辺りで Cantor 関数がさり気なく触れられています。 有名な外測度は、p.238 に äußeres Maß として出てきます。

話は変わって、欧米の大学には、honors program とか honors course というのがあって、数学で言えば、実用的な数学に対して理論的なそれを指すようです。
ただしその内容は、場所と人により様々。
実例をお目にかけると、 Honors Calculusとか。 これの積分が関係する章を見ると、そのカバーする内容・程度はこの数学演習のテキストと似たり寄ったりであることが見て取れるでしょう。
ただし、扱っている積分がいわゆるリーマン積分に限定されていて、その不自由さが全体の有用度を縛っているように見えます。
もう一つ今となっては古い本ですが、Jost の Postmodern Analysis (2005) を挙げておきましょう。
これはいろいろな意味で Dieudonne の Foundations of Modern Analysis を意識したもので、 リーマン積分不要論と叙述スタイルを踏襲しつつも、過度の抽象化は避け、微分方程式方面への接続を目指していると言えるでしょうか。
ただ、ブルバキの欠点であった一変数と多変数の積分のつながりの悪さは、そのまま引きずっているように見えます。
それがため、初等積分とルベーグ積分の間のギャップもそのまま抱え込んでいるということでもあり、postmodern になりきれていない印象を与えます。
新しい honors program の積分のあり方の試みの案とでもいうべきテキストとの違いがわかるでしょうか。

５月１７日

テキスト (monograph と呼んでおきます）を書いた理由は Introduction に述べてありますが、 皆さんのようにリーマン積分を本格的に学んだ人には、新たな混乱の種であったかと反省。
リーマンというよりはダルブーの与えた定義との関係を本文に入れ、それとリーマンの与えた定義が同じになることは、 付録にでも回すのがよい気がしてきました。
通常、不連続関数も積分できるように拡張したのがリーマンであるといった説明を目にしますが、 リーマンは単調関数が積分できることは述べてあるものの、連続関数が積分できることを（コーシーによる証明の不備を正す形で）最初に証明したのはダルブーでした。
リーマンが積分を定式化した当時は、一様連続性の認識がなかったのでした。 あるいは、連続関数が積分可能であることは、コーシーの定式化で既に終わっているとリーマンは思っていたのかもしれません。
いずれにせよ、monograph では、単調連続性（順序連続性）と単調列近似を全面に出すよう理論的な部分を書き改めました。
これはダニエルの与えた積分とも整合するもので、それの延長としてルベーグ積分がある意味自然に出てくる点を強調してあります。
ついでながら、monograph に載せた具体的な積分については、高木「解析概論」の積分関連をできるだけカバーするようにしてあります。もちろん、全部ではありません。
複素解析が関係するところは除いてありますし、特殊な話題も略してあります。
一方で、木に竹をつないだ感じの９章ルベーグ積分への接続が自然に行くような理論の構成を目指しました。

６月５日

重積分の定義で悩ましいところが、volume integral が well-defined であることの説明。
width integral の繰り返しで実現することで、この部分をやさしい１次元の場合に還元するという手法をテキストでは採用しましたが、 ついでに述べたテンソル積の部分が分かりづらいという指摘がありました。
テンソル積は、ベクトルについての基本的な操作ではあるのですが、どういう方法を取るにせよ、直感的ではなく、慣れるまでが多少の手間でしょうか。
これは、一般的なベクトル空間の場合も扱おうとするとどうも避けられないようですが、ベクトルを関数（関数空間）に限定すると、 比較的直接的な導入が可能です。
そのところの本文での説明が簡単に過ぎたところを補う形で、 付録に Functional Tensor Products の節を追加しておきました。修正・補充版からどうぞ。
こちらをまず読んで、それから本文の重積分のところを見直していただくのが良いかもしれません。

Dirichlet 積分を計算する例で、可積分かどうか怪しいと寝ぼけたことを言ってしまいましたが、 \[ \int_0^\infty dx\, \sin x \int_r^\infty e^{-xy}\,dy = \int_0^\infty e^{-rx} (\sin(x))/x\, dx \] で、右辺の積分が $\int_0^\infty e^{-rx}\,dx = 1/r$ で抑えられるのでした。
ということで淡々と計算すれば欲しい式が出てきます。
