\item 
べき級数 $\displaystyle \sum_{n=0}^\infty c_n z^n$ が $z = 1 + i$ で収束し、$z = -1+i$ で絶対収束しないとき、
その収束半径を求めよ。
% \item 
\end{enumerate}


\medskip
(i) テキストを見よ。

(ii) $z=1+i$ で収束することから、収束半径 $\rho$ は $|1+i| = \sqrt{2} \leq \rho$ を満たす。
もし、$\rho> \sqrt{2}$ であれば、 冪級数は収束半径の内部にある $z = -1 + i$ で絶対収束することになって、
矛盾を生じる。したがって、$\rho = \sqrt{2}$ である。

\noindent$\boxed 2$
\begin{enumerate}
\item 
複素関数 $\log(1+z)$ ($|z| < 1$) と $(1 + z)^{-1/2}$ ($|z| < 1$)のべき級数に
よる定義式を述べよ。
\item
複素関数
\[
-\frac{\log(1-z)}{\sqrt{1-z^2}}
\]
のべき級数表示を $z$ について4次の項まで求めよ。
\end{enumerate}


\medskip
(i) テキストを見よ。

(ii) 
\[
-\log(1-z) = z + \frac{1}{2}z^2 + \frac{1}{3}z^3 + \frac{1}{4}z^4 + \dots
\]
\[
\frac{1}{\sqrt{1-z^2}} = 1 + \frac{1}{2} z^2 + \frac{3}{8}z^4 + \dots 
\]
の積を計算し $z$ のべきで整理すると、
\[
-\frac{\log(1-z)}{\sqrt{1-z^2}} = z + \frac{1}{2}z^2 + \frac{5}{6} z^3 + \frac{1}{2} z^4 + \dots
\]
という表示を得る。

\medskip
[復習問題２]
複素関数 $\cos z$, $\sin z$ のべき級数を使った定義を述べ、
\[
\frac{\sin z}{(\cos z)^2}
\]
のべき級数表示を $z$ の5次の項まで求めよ。
